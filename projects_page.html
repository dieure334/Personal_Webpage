Basic Coloring App (Missing)

Programming Language/Platform: MIT App Inventor 2 (Android)

Year Created: 2018

This was an Android app that I created in high school. It is a basic coloring app designed for Android devices. How it works is that you would start off with a blank canvas on the screen and choose your color and pen size. The touchscreen feature of the device would allow you to draw on the screen. In addition to this, there were other features included such as the ability to erase and clear the canvas.

Unfortunately, this project was lost as I had created it using an account that used my high school email address. Once I graduated high school, the account was deleted, along with all projects I made using the account. I do have some screenshots and a single video I took of the program working, but the actual code is lost.

A-Frame VR Shooter Project

Programming Language/Platform: HTML, JavaScript, A-Frame (Google Cardboard/PC)

Year Created: 2019

This was a project I created with a friend for a Computer Science class during my Senior year of high school. This program uses HTML with JavaScript along with A-Frame to create a 3D VR environment. The program allows the user to "shoot" projectiles within the environment via click of the mouse. The project is originally meant to be compatible with Google Cardboard but works just as well on a computer. It also includes royalty-free music from Bensound.
